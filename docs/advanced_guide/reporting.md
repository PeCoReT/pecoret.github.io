# Reporting
You can create multiple reports in your project.
The reports can have different variants. For example, PDF or CSV.

Reports may contain an evaluation and recommendation section in the executive summary.
A change or version history of the report can be included too.

## Variants
A short overview of the current available variants is given below:

- **Pentest PDF:** A PDF that usually contains a management summary, project and finding details.

- **CSV Export:** A comma separated list of findings.


## Report Templates
Reports are derived from report templates.
Report templates can be deployed using python packages which provides full flexibility to create your unique reports.

If you want to learn how to create new report templates, have a look at the [customization pages](/docs/advanced_guide/customizations/report_templates.md).

By default, a simple example template is included.
